# Ingestion Source Strategy Decisions Now Promoted

Date: 2026-03-22

This note extracts the parts of the original ingestion source strategy discussion that have now been promoted into subject specs.

## Items Now Captured In `.spec`

- source-level origin metadata and display metadata
- source-level segment and timestamps
- content fingerprinting as a future-compatible requirement
- source identity separated from chunk identity
- preservation of structured markdown frontmatter as source-level metadata
- unified ingestion contract for multiple source types
- capture and inbox as source-entry workflow

## Purpose

The original strategy note still contains useful future-facing guidance, but these items are no longer only architectural discussion. They are now represented in `.spec` as current planning contract.
